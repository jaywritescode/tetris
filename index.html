Tetris:

which keys to mash

  • the left and right arrow keys do exactly what you'd expect them to do
  • the space bar and the down arrow key do a soft drop. there's no hard drop key
  • the Z and X keys rotate the current tetromino counter-clockwise and clockwise, respectively
  • the P key pauses the game. so does the "Pause Game" button. so does clicking outside of the game
